# Installation

This guide explains how to install and update the indiayz SDK.

---

## Requirements

- Python 3.9 or newer
- Active internet connection (backend APIs are hosted)

---

## Install

Use pip to install indiayz:

pip install indiayz

---

## Upgrade

To upgrade to the latest version:

pip install --upgrade indiayz

---

## Verify Installation

After installation, verify it works:

python -c "import indiayz; print(indiayz.__version__)"

If a version number is printed, installation was successful.
