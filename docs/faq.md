# FAQ

## Is indiayz free?

Yes. indiayz is open-source and free to use.

---

## Does indiayz run models locally?

No. indiayz uses hosted backend services.

---

## Can I use this commercially?

Yes. Apache 2.0 license allows commercial use.

---

## Is indiayz bundled in the SDK?

No. indiayz runs only on the backend.
